# タグ体系

## 共通タグ（全記事に付与）

| タグ | 用途 |
|-----|------|
| `#AIBizDev` | シリーズ統一タグ（メイン） |
| `#ノバセルBizDev` | ブランドタグ |
| `#AI事業開発` | 検索流入用（日本語キーワード） |
| `#BizDev公開日誌` | シリーズ名タグ |

---

## カテゴリ別タグ（記事テーマに応じて1-2個選択）

### AI技術の実務理解
- `#生成AI`
- `#AI活用`
- `#AIエージェント`
- `#プロンプト設計`

### フレームワーク分析
- `#事業戦略`
- `#競争戦略`
- `#ビジネスフレームワーク`

### 組織・人材論
- `#組織変革`
- `#AI人材`
- `#チームビルディング`

### AI×ビジネスモデル
- `#業務効率化`
- `#DX`
- `#新規事業`

---

## タグ選定ルール

1. **5つまで**: note仕様の上限
2. **共通タグ2つ + カテゴリタグ2-3つ** が基本構成
3. **note内で検索されやすい人気タグを優先**（独自造語タグは避ける）
4. **記事内容と一致するタグのみ**（関連性の低いタグはマイナス評価）
