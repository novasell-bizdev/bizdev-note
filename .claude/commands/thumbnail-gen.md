# サムネイル生成

note記事のサムネイル用プロンプト/デザイン指示を3パターン生成するスキル。

## 引数
$ARGUMENTS — 記事テーマ、タイトル、またはドラフトのファイルパス

## 固定仕様

| 項目 | 値 |
|------|-----|
| サイズ | **1280 × 670 px**（note推奨の1.91:1比率） |
| 共通要素 | シリーズロゴ「BizDev朝会」+ 連番 **#N** |
| フォント | ゴシック体 or サンセリフ体（可読性優先） |
| テキスト | 最大3行、合計30文字以内 |

## カテゴリ別カラースキーム

| カテゴリ | メインカラー | アクセント |
|---------|-----------|----------|
| AI技術の実務理解 | ブルー (#2563EB) | ホワイト |
| フレームワーク分析 | グリーン (#059669) | ホワイト |
| 組織・人材論 | アンバー (#D97706) | ダークブラウン |
| AI×ビジネスモデル | パープル (#7C3AED) | ホワイト |

## 実行手順

### 1. 入力解析

- 記事テーマ・タイトル・カテゴリを特定
- `published/content-registry.json` から次の #N を取得

### 2. 3パターンのデザイン指示を生成

#### パターンA: テキスト中心型

- 背景: カテゴリカラーのグラデーション
- 中央: タイトルテキスト（太字、白文字）
- 左上: シリーズロゴ「BizDev朝会」
- 右下: #N バッジ
- 用途: シンプルで統一感のあるデザイン

#### パターンB: アイコン型

- 背景: カテゴリカラー（ベタ塗り）
- 中央: テーマを象徴するアイコン/イラスト（ミニマルな線画）
- 下部: タイトルテキスト
- 左上: シリーズロゴ + #N
- 用途: 視覚的にテーマが伝わるデザイン

#### パターンC: 写真素材型

- 背景: テーマ関連のフリー素材写真（オーバーレイあり）
- オーバーレイ: カテゴリカラーの半透明帯
- テキスト: タイトル（白文字、ドロップシャドウ付き）
- 左上: シリーズロゴ + #N
- 用途: 臨場感のあるデザイン

### 3. 提案

各パターンについて以下を提示:

```
## サムネイル提案

### パターンA: テキスト中心型
- レイアウト: [ASCII図]
- カラー: メイン #XXXXXX / アクセント #XXXXXX
- テキスト: "..."
- Canva/Figma向けプロンプト: "..."
- 画像生成AIプロンプト: "..."

### パターンB: アイコン型
...

### パターンC: 写真素材型
...

**おすすめ**: パターンX
**理由**: ...
```
